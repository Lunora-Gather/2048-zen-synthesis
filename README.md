# 2048

一个基于 HTML、CSS 和 JavaScript 的网页版 2048 游戏，可以直接在浏览器中游玩。

**在线游玩：** https://lunora-gather.github.io/2048/

---

## 项目简介

2048 是一个经典数字合并游戏。玩家通过上下左右移动棋盘上的数字方块，让相同数字相撞并合并成更大的数字。游戏目标是在有限空间内尽可能合成更大的数字，并挑战更高分数。

本项目是一个纯前端静态网页项目，不依赖后端服务，也不需要复杂构建流程。代码主要由 `index.html`、`style.css` 和 `script.js` 组成，并额外加入了体验优化文件，用于改善布局、可访问性、移动端体验和运行稳定性。

---

## 当前功能

### 基础玩法

- 支持经典 2048 数字合并规则；
- 支持 4 × 4、5 × 5、6 × 6 三种棋盘大小；
- 支持当前分数和最高分记录；
- 支持最大方块统计；
- 支持游戏时间和移动步数统计；
- 支持重新开始游戏；
- 支持撤销上一步操作；
- 游戏结束和达成 2048 时有提示弹层。

### 操作方式

- 电脑端支持键盘方向键；
- 手机端支持触摸滑动；
- 支持点击按钮重新开始或撤销；
- 支持打开侧边栏查看成就和操作消息。

### 页面体验

- 支持浅色 / 深色模式；
- 支持多种视觉主题；
- 支持音效开关和音效预设；
- 支持桌面端和移动端响应式布局；
- 右侧信息面板与左侧游戏主体保持对齐；
- 操作消息区域使用内部滚动条，不会因为消息增多而撑开整个页面；
- 页面禁用 JavaScript 时会显示提示；
- 对键盘焦点、辅助标签和减少动画模式做了基础优化。

- Windows / Linux：`Ctrl + F5`
- macOS：`Command + Shift + R`

## 本地运行

由于项目是纯静态网页，最简单的方式是直接打开 `index.html`。

更推荐使用本地静态服务器运行，避免浏览器对本地文件的限制：

```bash
git clone https://github.com/Lunora-Gather/2048.git
cd 2048
npx -y http-server -p 8080 -c-1
```

然后打开：

```text
http://localhost:8080
```

如果没有安装 Node.js，也可以使用其他任意静态服务器，例如 VS Code 的 Live Server 插件。

## 主要文件说明

### `index.html`

页面入口文件，负责加载样式、脚本和页面结构。当前页面标题、描述、分享信息和在线地址都在这里配置。

### `style.css`

主要视觉样式文件，包含棋盘、方块、按钮、分数卡片、侧边栏、弹窗、响应式布局等样式。

### `script.js`

游戏核心逻辑文件，包含棋盘初始化、数字移动、数字合并、得分计算、胜负判断、存档、音效和操作日志等功能。

### `optimizations.css`

后加的体验优化样式文件，主要用于控制右侧信息面板、消息滚动、键盘焦点、减少动画、移动端适配等。

### `optimizations.js`

后加的运行时优化文件，不直接重写游戏核心规则，只补充辅助功能，例如页面隐藏时降低动画消耗、离开页面前保存游戏、增强可访问性属性等。

# 2048 · Zen Synthesis

一个精致的网页版 2048 游戏。项目保留经典数字合并规则，并加入多棋盘尺寸、主题切换、音效预设、撤销、统计、成就与移动端适配。

**在线游玩：** https://lunora-gather.github.io/2048/

---

## 项目概览

2048 是一个经典数字合并益智游戏。玩家通过上下左右移动棋盘，让相同数字方块合并成更大的数字，并在有限空间中挑战更高分数。

本项目是一个纯前端静态网页项目，不依赖后端服务，也不需要构建流程。代码主要由 `index.html`、`style.css`、`script.js` 与一组优化文件组成，可以直接部署到 GitHub Pages。

相比基础版 2048，本项目更强调完整的浏览器游戏体验：

- 更适合桌面端和移动端的响应式布局；
- 更清晰的分数、步数、时间与最大方块统计；
- 更丰富的视觉主题与深浅色显示模式；
- 更有反馈感的音效预设和活动日志；
- 更完整的成就系统与游戏信息抽屉。

## 功能亮点

| 分类 | 功能 |
| --- | --- |
| 基础玩法 | 标准 2048 移动、合并、胜利和失败判断 |
| 棋盘尺寸 | 支持 `4 × 4`、`5 × 5`、`6 × 6` |
| 操作方式 | 支持键盘方向键和移动端触摸滑动 |
| 撤销系统 | 支持撤销上一步操作，并保留多步历史状态 |
| 分数统计 | 显示当前分数、最高分、最大方块、移动步数和游戏时间 |
| 视觉主题 | 支持 Purple、Classic、Fresh、Minimal 等主题 |
| 显示模式 | 支持浅色 / 深色模式切换 |
| 音效预设 | 支持 Default Sound、Electronic、Retro、Mechanical |
| 成就系统 | 包含 First Merge、Reach 64、Reach 512、Reach 2048 |
| 活动抽屉 | 右侧面板集中展示成就和操作日志 |
| 页面体验 | 支持无 JavaScript 提示、基础可访问性和移动端布局优化 |

## 本地运行

由于项目是纯静态网页，最简单的方式是直接打开 `index.html`。

更推荐使用本地静态服务器运行，避免浏览器对本地文件的限制：

```bash
git clone https://github.com/Lunora-Gather/2048.git
cd 2048
npx -y http-server -p 8080 -c-1
```

然后打开：

```text
http://localhost:8080
```

也可以使用 VS Code 的 Live Server 插件或其他任意静态服务器。

```text

├── .nojekyll              # GitHub Pages 静态发布标记
```

## 设计方向

- **清晰聚焦**：棋盘区域始终是主视觉中心，设置和成就信息收纳在辅助区域。
- **轻量精致**：用字体、卡片、渐变、主题和动效提升质感，而不增加复杂依赖。
- **移动友好**：支持触摸滑动，并对移动端视口和布局做基础优化。
- **可维护**：核心逻辑集中在 `script.js`，优化代码以非侵入方式补充体验。
